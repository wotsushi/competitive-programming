## C
カテゴリ: 探索

最初のK回以内の移動だけ例外と考え、K個以下の部屋を開く→K回まで移動する という操作の列とみなすと
2回目以降の移動において閉じている部屋の存在は気にしなくてよいことが分かる。

初期位置からK回以内の移動で到達可能な部屋の集合をTとする。
Tに属す部屋のうち、最小のx, y座標を X, Y, 最大のx, y座標を X', Y' とする。
解は 1 + ceil(min(X - 1, Y - 1, W - X, H - Y) / K) である。
