# AGC031

## A
カテゴリ: 数学

各アルファベットの出現回数をA, B, .., Zとする。
解は ((A + 1)(B + 1)...(Z + 1) - 1) % (10**9 + 7)である。

## B
カテゴリ: 動的計画法

MOD = 10**9 + 7 とする。
漸化式dp[i]を以下で定義する。

* i = N ならば dp[i] = 1
* i < N かつ k > i + 1, C_i = C_k なるkが存在するならば dp[i] = (dp[i + 1] + dp[k]) % MOD
  * そのようなkが複数存在場合、最小のkとする
* それ以外ならば dp[i] = dp[i + 1]

解は dp[1] である。
