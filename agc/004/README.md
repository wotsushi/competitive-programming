## B
カテゴリ: 全探索
変色させる回数を0からN-1まで決め打ちしたときの必要時間の最小値を求め、
さらにそれらの最小値を解とすればよい。

変色させる回数をkで決め打ちしたとき、必要時間の最小値は、以下で求められる。
色iのスライムの獲得に必要な最小時間は min(a_{i - k} ,a_{i - k + 1}, ..., a_i) である。
各色について必要な最小時間の和Sを求める。
全体の必要時間の最小値は S + kx である。
