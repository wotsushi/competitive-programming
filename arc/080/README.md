# ARC080

## C
カテゴリ: 考察

a_i % 2 = 1 を満たす i の数を X, a_i % 2 = 0, a_i % 4 = 2 を満たす i の数を Y, a_i % 4 = 0 を満たす i の数を Z とする。
X <= Z または X = Z + 1, Y = 0 ならば、かつ、そのときに限り 'Yes' である。
