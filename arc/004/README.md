## B
カテゴリ: 幾何

d を降順 d' (d'_0 > d'_1 > ... > d'_{N - 1}) に並び替えると分かりやすい。
d'_0 <= d'_1 + d'_2 + ... + d'_{N - 1} ならば点0と点Nは同一点にできる。
d'_0 > d'_1 + d'_2 + ... + d'_{N - 1} ならば点Nは点0から
d'_0 - (d'_1 + d'_2 + ... + d'_{N - 1}) だけ離れた位置にプロットするのが
最適である。
解は、 max(0, 2 * max(d) - (d_0 + d_1 + ... + d_{N - 1})) である。
