## A
カテゴリ: 考察
y が奇数または負ならば行けないので、yは非負の偶数であることを仮定する。
k = y / 2 とする。
|a| <= k かつ (a - k) % 2 == 0 ならば k回で移動可能。
そうでなければ行けない。

（選んだ頂点をiとする）。
次に、頂点iを根とする木において、写真が存在する頂点のうち、深さが最大の頂点を一つ選ぶ
（選んだ頂点をjとする）。
