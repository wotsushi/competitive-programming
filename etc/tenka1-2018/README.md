## D
カテゴリ: 構築

N = 1 + 2 + ... + k を満たすkが存在するならば Yes である。
このとき、以下のように構築すればよい。

* S_1 = {1, 2, ..., k}
* S_2 = {1, k + 1, ..., 2k - 1}
* S_3 = {2, k + 1, 2k, ..., 3k - 3}
* S_4 = {3, k + 2, 2k + 1, 3k - 2, ..., 4k - 6}
* ...
* S_k = {k - 1, 2k - 2, 3k - 3, ..., k(k + 1) / 2}
* S_{k + 1} = {k, 2k - 1, 3k - 3, ..., k(k + 1) / 2}
