## C
カテゴリ: 全探索

商品の購入個数の候補値は floor(X / 105) 以上 floor(X / 100) 以下である。
各候補値kについて、少なくとも100k円は払うことになる。購入する商品よって追加で0円以上5円以下を支払うとみなす。
すると、X - 100k が0以上5k以下であればちょうどX円となるような商品の購入の仕方が存在することが分かる。
