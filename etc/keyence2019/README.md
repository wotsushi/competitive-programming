## D
カテゴリ: 数え上げ

a(k) を A_i = k を満たすiの個数とする。b(k) を B_i = k を満たすiの個数とする。

W_k, H_k, P_k を以下で定義する。

* k = NM + 1 ならば
  * W_k = 0
  * H_k = 0
  * P_k = 1
* a(k) >= 2 または b(k) >= 2 ならば
  * W_k = 0
  * H_k = 0
  * P_k = 0
* a(k) = b(k) = 1 ならば
  * W_k = W_{k + 1} + 1
  * H_k = H_{k + 1} + 1
  * P_k = P_{k + 1}
* a(k) = 1, b(k) = 0 ならば
  * W_k = W_{k + 1}
  * H_k = H_{k + 1} + 1
  * P_k = P_{k + 1} W_{k + 1}
* a(k) = 0, b(k) = 1 ならば
  * W_k = W_{k + 1} + 1
  * H_k = H_{k + 1}
  * P_k = P_{k + 1} H_{k + 1}
* a(k) = b(k) = 0 ならば
  * W_k = W_{k + 1}
  * H_k = H_{k + 1}
  * P_k = P_{k + 1} (W_{k + 1} H_{k + 1} - (NM - k))

解は、 P_1 である。
