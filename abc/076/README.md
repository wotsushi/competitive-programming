## D
カテゴリ: 貪欲法

V_i を以下で定義する。

* V_{N + 1} = 0
* V_i = min(v_i, V_{i + 1} + t_i)

T_i を以下で定義する。

* T_0 = 0
* T_i = T_{i - 1} + t_i

最適な走り形における時刻 t' の速度を f(t') とすると、

* t' = 0 ならば f(t') = 0
* T_i < t' <= T_{i + 1} ならば min(v_{i + 1}, f(T_i) + (t' - T_i), V_{i + 1} + (T_{i + 1} - t'))

が成り立つ。
fを時刻 0から 時刻 T_N で積分した値が解である。
