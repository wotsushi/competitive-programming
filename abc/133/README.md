## D
カテゴリ: 数学

山iに降った雨の量を 2x_i で表す。
x_i + x_{i + 1} = A_i が成り立つ。
よって、 x_1 = (A_1 - A_2 + A_3 - A_4 + ... + A_{N}) / 2 である。
x_2, ..., x_N は x_1 をもとに求めればよい。
