## E
カテゴリ: 動的計画法

Sの先頭からi文字目までの部分文字列S[i]とTの先頭からj文字目までの部分文字列T[j]についての
解をdp[i][j]とする。
dp[i][j]について、以下が成り立つ。

* i = 0 または j = 0 ならば dp[i][j] = 1
* S_i = T_j ならば dp[i][j] = 2dp[i - 1][j - 1] + dp[i - 1][j] + dp[i][j - 1] - 2dp[i - 1][j - 1] = dp[i - 1][j] + dp[i][j - 1]
* そうでなければ dp[i][j] = dp[i - 1][j] + dp[i][j - 1] - dp[i - 1][j - 1]

解は dp[N][M] である。
