## C
カテゴリ: 貪欲法

N個のコマは X_i に等しい座標に置くべきであることに注意。
Xは昇順ソートされていると仮定する。
まず、X_1 にコマをおいて損はしない。
他コマについて、X_{i} - X_{i - 1} が大きいiから順にコマを置けばよい。

## D
カテゴリ: 考察

Xの各ビットごとに0をとるべきか1をとるべきかをAの値に応じて決めればよい（多数決させるイメージ）。
KのMSBを1とする場合、Kの値によっては下位ビットを1にできない場合があるので、
MSBを0とする場合と1とする場合で場合分けして最適値を求める。
