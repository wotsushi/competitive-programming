## D
カテゴリ: 数え上げ

移動回数kは (X + Y) / 3 となる。ここで、X + Y が3の倍数でない場合、0通りとなる。
(i + 1, j + 2)の移動を実行する回数をaとすると、 a = X - k が成り立つ。
(i + 2, j + 1)の移動を実行する回数をbとすると、 b = Y - k が成り立つ。
a >= 0 かつ b >=0 でなければ0通りである。
そうでなければ kCa を出力すればよい。
