## D
カテゴリ: 数学

レベルNバーガーの層の総数は 2^{N+2} - 3 である。
レベルNバーガーのパティの総数は 2^{N+1} - 1 である。
解を f(N, X) で表す。
以下が成り立つ。

* X <= 0 ならば f(N, X) = 0
* X >= 2^{N + 2} - 3 ならば f(N, X) = 2^{N + 1} - 1
* 1 <= X <= 2^{N + 2} - 4 ならば f(N - 1, X - 1) + f(0, X - (2^{N + 1} - 2)) + f(N - 1, X - (2^{N + 1} - 1))
