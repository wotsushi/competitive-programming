## E
カテゴリ: 構築

頂点1を中心とするスターグラフをGとする。
Gの最短距離が2であるような頂点対 (i, j) は (N - 1)(N - 2)/2 個存在する。
K が (N - 1)(N - 2)/2 より大きい場合、条件を満たすグラフは構築できない。
そうでない場合、 G に (N - 1)(N - 2)/2 - K 本の辺を追加すればよい（追加する辺は適当でよい）。
