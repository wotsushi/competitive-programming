## D
カテゴリ: 貪欲法

閉じられていない '(' の個数だけ S の末尾に ')' を付与すればよい。
閉じられていない ')' の個数だけ S の先頭に '(' を付与すればよい。

1 <= i <= N について S の先頭から i番目の文字を S_i, 文字列 S_1 S_2 ... S_i を S[:i] で表す。
S[:i] の閉じられていない '(' の個数を f(i), S[:i] の閉じられていない ')' の個数を g(i) で表す。
便宜上、f(0) = g(0) = 0 と定義する。このとき、

* S_i = '(' ならば
  * f(i) = f(i - 1) + 1
  * g(i) = g(i - 1)
* S_i = ')' かつ f(i - 1) = 0 ならば
  * f(i) = 0
  * g(i) = g(i - 1) + 1
* S_i = ')' かつ f(i - 1) >= 1 ならば
  * f(i) = f(i - 1) - 1
  * g(i) = g(i - 1)
