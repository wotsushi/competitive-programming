## D
カテゴリ: 貪欲法

寿司をおいしさが大きい順にソートする。
まず、大きい寿司から順にK個食べてみる。
次に、食べた寿司について、重複するものがあればそれを食べず、おいしさが(K + 1)番目以降のもので食べてない寿司を食べた場合の
満足ポイントを求める。
同様に、種類ボーナスポイントを増やした場合の満足ポイントを求め、それらのうち最大のものを解とすればよい。
